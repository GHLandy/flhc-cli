## Install

```bash
npm i -g flhc-cli@git+https://git@github.com/GHLandy/flhc-cli.git

# Or install the dev branch
npm i -g flhc-cli@git+https://git@github.com/GHLandy/flhc-cli.git#dev
```

## Usage

```bash
# For help details
flhc

# Sync vimrc to ~/.vimrc
flhc vim
```
